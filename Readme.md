# Fake Taobao Deeplink

复制 ** com.taobao.tao.welcome.Welcome ** 的intent-filter来拦截误触广告后自动拉起淘宝app

***

> Bilibili 优酷 芒果TV可以拦截；斗鱼TV拦截后无法正常进入应用需要手动点跳转
>
> 微博会判断拉起淘宝是否成功，第一次拦截后下一次跳转淘宝会改用显示Intent拉起，**需要使用手机系统自带的应用双开避免**
>
> 咸鱼的淘宝商品也可以拦截
>
> Chrome跳转无法拦截
